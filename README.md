# 🐾 Doggies - Panel Administrativo 

Panel administrativo desarrollado con **React JS + Vite** como ejercicio práctico.

## 📋 Descripción

Sitio web que simula un panel administrativo con navegación. Permite visualizar y gestionar Clientes, Proveedores y Usuarios del sistema mediante una interfaz limpia y responsiva.

## 🚀 Tecnologías utilizadas

- React 
- Bootstrap como framework de estilos
- CSS personalizado con variables y diseño propio

## 🛠️ Pasos seguidos

1. **Inicialización del proyecto** con Vite y React:
```bash
   npm create vite@latest
   npm install
```

2. **Instalación de dependencias** necesarias:
```bash
   npm install react-router-dom
   npm install bootstrap
```

3. **Configuración del entorno** en Visual Studio Code.

4. **Creación de la estructura de componentes:**
   - `Navbar.jsx` con logo y barra de navegación
   - `Header.jsx` con el logo de la marca
   - `Menu.jsx` con los enlaces de navegación usando `<NavLink>`

5. **Implementación del sistema de navegación** con `<BrowserRouter>`, `<Routes>` y `<Route>` en `App.jsx`.

6. **Creación de las vistas:**
   - `Clientes.jsx` — tabla con nombre, correo, teléfono y ciudad
   - `Proveedor.jsx` — tabla con nombre, contacto, teléfono y ciudad
   - `Usuarios.jsx` — tabla con nombre de usuario, rol y estado
   - `Logout.jsx` — vista de cierre de sesión

7. **Aplicación de estilos** con Bootstrap como framework base y CSS personalizado usando variables para colores, tipografía y componentes como badges y tablas.

8. **Subida del proyecto a GitHub** con Git:
```bash
   git init
   git add .
   git commit -m "primer commit - proyecto Doggies"
   git push -u origin main
```
